Importancia de la keyword

La url (uniform resource locator) es una dirección que designar un recurso en internet es el identificador “dirección” que permite una página entre los miles de millones de recursos que componen la web La url es uno de los elemento visible en el navegador por lo tanto la url informan directamente al usuario del contenido de la página para que acceda fácilmente Para un mejor keyword en la url podemos hablar de las siguiente detalles que deben de llevar Escoger una palabra en la url que cubra el tema tratado en su página y debe de ser simple Claro y legible guiones (-) permitido

Meta robots

Nos permite proporcionar instrucciones para indexación de una página web esto se logra con la etiqueta meta “robots”, la búsqueda estará disponible en el momento que el navegador analice la página

index:

Esta directriz permite al robot de búsqueda la indexación inmediata a la web Ejemplo

noindex:

Este tag prohíbe al buscador transferir contenido de una página HTML a su base de datos es decir el atributo “robots” se dirige a todos los buscadore y “noindex” indica que no se permite la indexación Ejemplo

Follow:

permite a los motores de búsqueda un comportamiento estándar que consiste en recorrer los enlaces que se encuentran en una página HTML es decir analiza todo los enlaces que tenga la página Ejemplo

Robots.txt

Robots.txt son archivos utilizados para favorecer la navegación de un algoritmo de búsqueda en un sitio web ya que ayuda a controlar el acceso a cierta información y su especialidad es dar órdenes específicas para cada robots de búsqueda, Los archivos robots.txt constan de una o varias reglas. Cada regla bloquea o permite el acceso de un determinado rastreador a una ruta de archivo en el marcado en tu archivo robots.txt utilizando el comando user-agent Para conocer el nombre de cada user-agent, puedes consultar la base de datos de web robots que muestra los principales motores de búsqueda de mercado Por ejemplo el robots de búsqueda principal de google es google bot Si se le quiere dar órdenes especifica el comando que ingresó en tu archivo robots,txt seria user-agent: googlebot También se quieres dejar ordenes para el robots dejar ordenes para el robots del navegador bing seria

user-agent: *

En caso que deseamos ingresar la dirección a los navegadores en general simplemente reemplaza el nombre con un asterisco (*) ejemplo user-agent: *

user-agent:

significa ingresar la dirección de la página después del comando

Comando disallow:

Es el encargado de escribir que la página o sitio web debe incluirse en los resultados de búsqueda

Comando allow:

Se recomienda utilizar el comando allow cuendo se quiera bloquear un archivo o directorio si se quiere bloquear el acceso por ejemplo a la carpeta “archivo” pero necesitas permitir el acceso a la página productos.php con el siguiente comando disallow:/archivo/ allow:/files/productos
